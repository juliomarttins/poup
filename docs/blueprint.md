# **App Name**: Family Finance Manager

## Core Features:

- Family Account Management: Allow family members to log in via email and invite other members to manage the shared finance.
- Detailed Transaction Logging: Enable logging and categorizing of income, expenses, and debts for detailed financial tracking.
- Financial Dashboard: Provide a financial overview with reports and debt-focused charts.
- Debt Management Tool: AI powered tool to offer a recommendation plan to pay the debts based on the current incomes and outcomes.
- Personalized Dashboard: Allow users to define the set of visual information shown at the dashboard
- Color scheme customization: Allow users to choose color pallet and toggle between light and dark modes

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), providing a sense of trust and stability.
- Background color: Light gray (#F0F4F7) for light mode and dark gray (#37474F) for dark mode, offering comfortable contrast.
- Accent color: Soft green (#81C784) to highlight key actions and positive financial indicators.
- Body and headline font: 'Inter', a sans-serif, will be used for clear and modern readability throughout the app.
- Use clean, professional icons in a consistent style to represent different categories and actions.
- Design a clean and intuitive layout that is easy to navigate on smaller screens. Implement a clear hierarchy with well-spaced elements for comfortable viewing.
